# Standard Table 标准二维表组件
- 管理平台中所有二维表的基础组件

------

## StandardTableProps 组件接受的参数

| 参数名 | 参数类型 | 参数说明 | 参数实例 |  默认值 |
| --- | --- | --- | --- | --- |
| title | string | ToolBar 列表工具栏标题 | "高级列表" | - |
| tooltip | string | ToolBar 列表工具栏标题之后会展示一个 icon，hover 之后提示一些信息 | "这是一个标题提示" | - |
